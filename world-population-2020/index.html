Population mondiale en 2020

Estimation du dénombrement de la population mondiale pour 2020 sous forme de grille. La grille a une résolution d'environ 110 km. Utilisez le curseur pour filtrer les cellules en fonction du nombre de personnes par unité de grille.

Données raster de population d'origine de SEDAC. Ce prototype a été réalisé ArcGIS API for JavaScript. Le code de cette application : Github.
